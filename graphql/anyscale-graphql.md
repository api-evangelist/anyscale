# Anyscale GraphQL API

Anyscale is a managed Ray platform for distributed AI compute. Their Endpoints API provides OpenAI-compatible access to open-source LLMs (Llama, Mistral), fine-tuning, and deployment.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Anyscale REST/gRPC API
**Documentation:** https://docs.anyscale.com/
